# teebets-site

Public marketing/legal site for the **Tee Bets** iOS app, served via GitHub Pages.

- `index.html` — landing page
- `privacy.html` — Privacy Policy (the App Store "Privacy Policy URL")
- `support.html` — Support page (the App Store "Support URL")
- `.nojekyll` — tells GitHub Pages to serve files as-is (no Jekyll build)

The canonical source of the privacy policy lives in the app repo at `docs/PRIVACY_POLICY.md`; the HTML
here is the published copy. Keep them in sync when the policy changes.

## Publishing

GitHub Pages is served from the `main` branch root. Live URLs:

- https://Bdnicholson.github.io/teebets-site/
- https://Bdnicholson.github.io/teebets-site/privacy.html
- https://Bdnicholson.github.io/teebets-site/support.html
